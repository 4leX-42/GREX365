Set-ExecutionPolicy

ERROR TÍPICO
You cannot run this script on the current system | not digitally signed
SOLUCION
1. Abrir PowerShell como administrador
2. Ejecutar:
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
3. En la misma ruta:
Unblock-File .\Main.ps1
4. Ejecutar:
.\Main.ps1
Modifica la política solo para el usuario actual
NOTA
PowerShell bloquea scripts por política de ejecución
Local permitido; .ps1 descargados requieren firma o desbloqueo

https://learn.microsoft.com/en-us/powershell/module/microsoft.powershell.security/set-executionpolicy
